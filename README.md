# How Hungry?
Modify hunger system as much as you want.
## About
Tired of vanilla hunger system or want to change it for your map/modpack? Well, this mod got you.
## Features
- Change if hunger does damage and when it does it
- Change if hunger does heal and when it does it
- Which effects apply to player if they're hungry
- Change amount of saturation needed to run (needed both on client and server)
# FAQ
- Can I use this mod in my modpack?  
Yes, of course!
- Can I rehost this mod on different mod platform?  
No. This mod is available on Modrinth and CurseForge. That's it.
- Ports to different mod loaders?  
Nah. At least not now. Currently only Fabric and Quilt are supported.
- Is it needed on both server and client?  
Actually no. If you want to change running hunger - it's handled on client for some reason. To do this change `requireOnClient` setting.
## Dependencies
This mod runs starting from 1.20+ and only needs [FabricAPI](https://modrinth.com/mod/fabric-api).
